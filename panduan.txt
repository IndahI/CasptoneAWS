cara make buka terminal dulu trus masukin perintah:
1. python -m venv venv
2. venv\Scripts\activate
3. pip install Flask pymongo dnspython certifi python-dotenv
4. py app.py

langkah pertama buat bikin virtual environment 
langkah kedua itu buat masuk ke virtual environmentnya
langkah ketiga itu install library python
langkah keempat itu running application

kalo mau ganti database mongoDB ke punya kalian gantinya di .env

kalo mau dipush di github dideploy saran gua mending masukkin ini ke .gitignore nya
